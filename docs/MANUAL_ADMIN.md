# Manual de Usuario — Administrador

## Funciones principales
- Ver y modificar cualquier caja, base y tope
- Registrar ajustes globales
- Ver todos los reportes y dashboard
- Exportar reportes
- Auditar movimientos y cierres de caja

## Flujos básicos
### 1. Ver y modificar cajas
- Selecciona usuario
- Consulta y ajusta saldos, bases, topes

### 2. Auditar movimientos
- Filtra por usuario, fecha, tipo de movimiento
- Exporta historial

### 3. Ver reportes globales
- Consulta cierre de caja, cartera activa, volados, microseguro
- Exporta a Excel/PDF

## Restricciones
- Puede realizar cualquier operación
