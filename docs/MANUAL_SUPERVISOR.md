# Manual de Usuario — Supervisor

## Funciones principales
- Ver y administrar caja propia y de cobradores asignados
- Definir base y tope de préstamo para cobradores
- Transferir base a cobradores
- Registrar retiros y cierre de caja
- Marcar clientes como volados
- Ver reportes y dashboard

## Flujos básicos
### 1. Transferir base
- Selecciona cobrador
- Ingresa monto y confirma

### 2. Registrar retiro
- Selecciona caja (propia o de cobrador)
- Ingresa monto y confirma

### 3. Marcar cliente como volado
- Selecciona cliente
- Ingresa monto bloqueado y observaciones

### 4. Ver reportes y cierre de caja
- Filtra por fecha, cobrador, tipo de movimiento
- Exporta a Excel/PDF

## Restricciones
- No puede modificar cajas de otros supervisores
- No puede retirar de microseguro de cobradores ajenos
