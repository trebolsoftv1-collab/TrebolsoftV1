# Manual de Usuario — Cobrador

## Funciones principales
- Ver saldo de caja y microseguro
- Registrar préstamos a clientes
- Registrar cobros, abonos y pagos totales
- Registrar gastos
- Ver movimientos y reportes
- Marcar observaciones en cobros

## Flujos básicos
### 1. Registrar préstamo
- Selecciona cliente
- Ingresa monto, plazo, cuotas, interés, microseguro
- Confirma y revisa proyección de cuotas

### 2. Registrar cobro
- Selecciona crédito y cliente
- Marca tipo de cobro (cuota, abono, pago total, no pago)
- Ingresa monto y observaciones

### 3. Registrar gasto
- Ingresa monto, categoría y observación

### 4. Ver reportes
- Filtra por fecha, tipo de movimiento, cliente
- Exporta a Excel/PDF

## Restricciones
- No puede retirar dinero ni modificar base
- No puede transferir ni modificar topes
- No puede marcar clientes como volados
